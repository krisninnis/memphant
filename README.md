# 🧠 Project Brain

A desktop app that lets you **store, manage, and transfer project memory between AI platforms** (ChatGPT, Claude, Grok, etc.)

---

## 🚀 What Problem It Solves

When switching between AI tools, you lose context.

Project Brain fixes that by:

- Saving structured project memory
- Allowing export/import between AI platforms
- Keeping decisions, goals, and progress consistent

---

## ✨ Features

- 📂 Project memory system (JSON-based)
- 🔄 AI handoff export/import
- 🧠 Structured memory (goals, rules, decisions, next steps)
- 📁 Project folder scanning (detects tech stack)
- 💾 Local desktop storage (Tauri)

---

## 🎯 Vision

Enable seamless cross-AI workflows without losing context.

---

## ⚠️ Current Status

Early development.

Main focus:

- Improving UX (making it simple for non-technical users)
- Adding validation layer
- Improving project scanning accuracy

---

## 🛠 Tech Stack

- React + TypeScript
- Tauri (desktop app)
- Rust (backend commands)

---

## 🧪 Example Use Case

1. Build a project in ChatGPT
2. Export memory
3. Open Claude or another AI
4. Paste/import memory
5. Continue seamlessly
